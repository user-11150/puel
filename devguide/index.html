    2. Makefile¶

    The Makefile have many commands. It's good for developers. For example. 

    make
    It's equals 
    rm -rf build
rm -rf dist
rm -rf .mypy_cache
python -m pip uninstall -y uel
python setup.py install
    It's very good. developrs always use it.

    Let's learn it.

    The all commands with the Makefile:

    commnad name information
    refrensh If you are a very small change. never use it. if you remove some c extension or change extra compile args, recommend using this
    lint The typing annotations analyzer
    install install(or reinstall) UEL
    clean remove dists, mypy caches, build caches
    report Report UEL source codes. please after make clean run this
    test unittests
    coverage coverage.py
    build source dist the uel
    upload upload the uel to PyPI
    docs-serve docs serve
    docs-build docs build
